A Beatsaber custom light saber mod that based on scene in "Cosmic Princess Kaguya!", light saber used in Kaguya's final live.  
Only compatible on PC, version that passed test is `1.42.2`, custom mod `CustomSaberLite` is required.  
一个节奏光剑自定义光剑Mod，来源于《超时空辉夜姬！》中辉夜的毕业live中使用的光剑。  
只能做PC上，在`1.42.2`上测试通过，需要`CustomSaberLite` Mod 才能运行。
